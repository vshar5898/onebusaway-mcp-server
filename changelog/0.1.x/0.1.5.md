# 0.1.5 — 2026-05-30

## Added

- **`enrichment` block** on `onebusaway_find_stops`, `onebusaway_find_routes`, `onebusaway_search_stops`, `onebusaway_search_routes`, `onebusaway_get_arrivals`, `onebusaway_get_vehicles`, `onebusaway_get_schedule_for_stop`, `onebusaway_get_schedule_for_route`, and `onebusaway_list_agencies` — each surfaces a typed `enrichment` block with query/filter echoes, result counts, and optional `notice` guidance when results are empty or truncated. Reaches both `structuredContent` JSON and the `content[]` markdown trailer.

## Changed

- **`@cyanheads/mcp-ts-core`** `^0.9.13` → `^0.9.16` — framework adoption enabling `ctx.enrich`, `ctx.enrich.notice`, and `enrichmentTrailer`.
- **`format` script** split into `format` (safe biome write) and `format:unsafe` (unsafe biome write) for explicit control.
